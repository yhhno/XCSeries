装饰者模式 样例 步骤
1.先定义一个抽象类Phone类,添加抽象类Show. 为什么添加show方法
2.定义一个ApplePhone类,继承抽象类Phone,实现show方法, 添加显示手机本身的信息
3.定义一个抽象类Decorate,继承抽象类Phone类,实现Show方法,添加phone字段,添加表示给手机转配的动作的SetPhone的方法.
4.定义表示具体装配的Sticker类和Accessories类,继承Decorate类,实现各自的show方法,表示各自的装配信息.
5.在main函数中调用.
 static void Main(string[] args)
        {
            //1.先有个手机
            Phone phone = new ApplePhone();

            //2.再有配件
            Decorate sticker = new Sticker();
            Decorate accessories = new Accessories();

            //3.先装配贴膜
            sticker.SetDecorate(phone);

            //4.在装配挂件,注意:是将挂件装配到已经贴膜的手机上
            accessories.SetDecorate(sticker);//如何实现 是将挂件装配到已经贴膜的手机上

            //5.显示下装配结构
            accessories.show();
        }
5.扩展: 新增个装配方式, 新加个子类继承Decorate类即可
